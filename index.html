Food Recognition and Calorie Measurement

Food Recognition and Calorie Measurement is an AI-driven nutrition detection system built using Python, Flask, and EfficientNetB4. The web app predicts food category and nutritional values from uploaded images, trained on a Kaggle dataset and tested on Colab. Project 4

Kidney Stone Detection

Kidney Stone Detection using YOLOv8x is an AI-powered diagnostic web application developed with Python, Flask, and a YOLOv8x model trained on a Kaggle dataset using Google Colab. The system detects kidney stones from uploaded medical images and displays predictions in real time. The trained model (kidneystonedetection.pt) is integrated into the Flask backend for seamless image processing. The responsive webpage features a center-aligned image upload section, a theme toggle switch, and real-time detection preview. Predicted images are saved in the backend, with options to view and download results. This application offers an efficient, user-friendly interface for kidney stone detection, combining deep learning with modern web design.
